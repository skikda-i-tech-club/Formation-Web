Introduction au Web

Qu'est ce qu'un système informatique ?

  • Son but est d'automatiser le traitement, le stockage, la transmission et la présentation de l'information.
  • Il est constitué d'une partie matérielle (hardware) et une partie logicielle (software).
  • Exemples: PCs, smartphones, tablettes, TVs, véhicules, automates programmables industriels, ...

Qu'est ce qu'un réseau informatique ?

  • Un ensemble d'ordinateurs connectés entre eux.
  • Son but, entre autres, est de faciliter le partage des ressources.
  • Chaque machine dans un réseau est identifiée par une adresse IP.
  • La forme d'une adresse IP : x.x.x.x où x est un nombre entre 0 et 255.
  • Exemples d'adresses : 127.0.0.1, 8.8.8.8, 192.168.1.1, 172.217.23.228, ...

Qu'est ce qu'Internet ?

  • C'est un réseau mondial (le réseau des réseaux).
  • A voir : Warriors of the net.

Qu'est ce que le Web ?

  • Abréviation de l'anglais World Wide Web (WWW), signifiant toile d'araignée mondiale.
  • Inventé le 12 mars 1989 au CERN, Tim Berners-Lee et Robert Cailliau.

Les ressources

  • Le web est composé de ressources (pages, images, sons, vidéos, ...).
  • Les ressources sont liées entres elles par des liens hypertextes (hypermedias).

Les ressources

URL

  • URL pour Uniform Resource Locator.
  • Chaque ressource est identifiée par une URL ou adresse web.
  • Les URL ont diverses formes :
    http://fr.wikipedia.org/
    ftp://ftp.rfc-editor.org/in-notes/rfc2396.txt
    mailto:skikdaitechclub@gmail.com
    file:///C:/hello_wolrd.js

HTTP

  • HTTP pour HyperText Transfer Protocol.
  • C'est le protocol qui permet le partage des ressources sur le web.

Serveur HTTP

  • Un serveur HTTP est le logiciel utilisé pour servir des ressources.
  • Exemple: Apache HTTP Server, NGINX, Apache Tomcat, NodeJS, ...

Client HTTP

  • Un client HTTP est le logiciel utilisé pour accéder aux ressources.
  • Tout les navigateurs web sont des clients HTTP.
